# 证据卡字段解读指南

`money evidence <标的> --format json` 输出一张证据卡。下面逐块说明每个字段的含义，以及**怎么把它读成多空信号**。记住：这些只是证据，最终判断是你的。

## asset
- `symbol` / `name` / `type`（otc_fund=场外基金，gold=黄金）。`as_of` 无则看 price.latest_date。

## price
- `latest` / `latest_date`：最新净值/价及日期。场外基金 T+1，latest_date 通常是昨天。
- `returns`：`1w/1m/3m/1y` 区间收益（小数，0.05=+5%）。多周期背离要留意——比如 1y 大涨但 1m 大跌，可能是高位回调。
- `max_drawdown_1y`：近一年最大回撤（负数）。数值越大（越接近 -0.3、-0.5）风险越高。
- `volatility_1y`：年化波动率。黄金/宽基通常 0.15~0.30，越高波动越剧烈。

## indicators
- `ma`：`ma5/ma10/ma20/ma60` 及 `alignment`。
  - `多头`（短期均线在上）= 上升趋势，偏多。
  - `空头`（短期均线在下）= 下降趋势，偏空。
  - `纠缠` = 无明确趋势，倾向观望。
- `macd`：`dif/dea/hist` 及 `state`。
  - `金叉`（hist 由负转正）= 动能转多的信号。
  - `死叉`（由正转负）= 动能转空。
  - `—` = 无新交叉；此时看 hist 正负判断当前动能方向。
- `rsi14`：相对强弱，0~100。
  - `>70~75` 超买（短期可能回调），`<30` 超卖（短期可能反弹），50 附近中性。
- `boll`：布林带位置 `pos`（上轨附近/中轨/下轨附近）与 `width`（带宽=波动）。
  - 触上轨=短期偏热，触下轨=短期偏冷；带宽骤缩常预示变盘。

## percentile
- `price_pct`：最新值在近 N 年的百分位（0~1，point-in-time 计算，无前视偏差）。
  - 接近 1 = 处于历史高位（偏贵，追高需谨慎）；接近 0 = 历史低位（偏便宜，但要分辨是"便宜"还是"下跌趋势中"）。

## news
- 近期舆情原始信息列表（title/source/date/url/summary）。**只是原始材料，未做判断**。
- 你要鉴别：来源是否可信、是否过时、对该标的是利好还是利空、是否只是营销软文。舆情往往不够新，务必用 WebSearch 补最新的。

## track_record
- 该标的历史信号的复盘统计：`past_signals`（累计预测数）、`hit_rate`（命中率）、`ic`（信息系数，信号强弱与实际涨跌的相关性，衡量预测力）、`vs_benchmark`（相对买入持有的超额）。
- 用它校准把握：历史上你对这类标的/形态判得准就可以自信些，判不准就收敛。`None` 表示还没有已到期的样本。

## data_quality（最重要的诚实性字段）
- `price` / `news`：`ok` / `stale`（陈旧）/ `partial`（不足一年，指标置信度低）/ `missing`（无数据）。
- `notes`：具体说明。
- **只要有 partial/missing，你必须在结论里明说，并相应下调置信度。** 拿残缺数据硬下结论是这套系统最忌讳的事。
