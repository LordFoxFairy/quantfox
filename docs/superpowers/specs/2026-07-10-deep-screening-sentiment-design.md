# 深度筛选 + 舆情分析 功能设计（草案）

> 状态：**设计草案**，未实现。后续由 skills 优化落地。
> 日期：2026-07-10 ｜ 来源：与用户 brainstorming
> 相关：`docs/lesson.md`(2026-07-10 看榜教训)、`docs/skill-improvements.md`(第4条)

## 1. 目标与用户背景

- **用户诉求**：找"靠谱 + 高概率正收益 + 收益高"的基金；持有取向 **短线 1-3 个月**。
- **本质**（用户原话）：找基金是为赚钱，希望 **把全市场尽量都看一遍**（全面），不是端一个按收益排好的现成榜单看前几名。
- **现状痛点**：
  - `screen.py` 打分 = 多周期收益加权 percentile → 榜首必然是近期暴涨的过热单一板块（实测前12全是半导体/AI，近3月翻倍、52周最高点），直接用=看榜，主题高度集中。
  - 项目**无舆情能力**（akshare 有资讯/资金流接口，但未接）。

## 2. 架构：四层漏斗

```
Layer 0  全量拉取     load_universe(多类型合并) → ~14000 只（已验证 13,905）
Layer 1  量化基本面   多因子分层：每板块选优 → 去过热 → 去集中 → A/C去重
Layer 2  舆情层       资讯 + 重仓股 + 资金面/热度，加权合成舆情分
Layer 3  精筛验风险   短名单逐只拉 evidence 证据卡验风险
最终     综合排序     量化 : 舆情 加权（可调），输出分散短名单 + 报告
```

### Layer 1 · 量化基本面多因子（已跑通原型）
在全市场全量上自造因子，而非按收益排名：
- **已确立赢家**：近1/2/3年 percentile（趋势成立，非昙花一现）
- **动能仍在**：近3/6月 > 0 且近1年 > 0
- **反过热**：剔除近3月暴涨（如 >55%，抛物线=血崩前兆）
- **回调不追高**：近1周温和、近1月 < 近3月×0.6（蓄势/回调而非顶）→ 呼应"顶准别顶在山顶"
- **稳健**：多周期（1m/3m/6m/1y）为正的个数
- **去集中**：**A/C 份额去重**（同名去尾缀保最优）+ **每主题限流**（如每板块≤2只），强制分散
- **每板块选优**：按主题标签（科技/半导体、医药、消费、新能源、军工、金融、红利价值、宽基、海外/商品…）分组，先各组选优再汇总

### Layer 2 · 舆情层（用户定：混合加权）
三路信号加权合成舆情分（权重可调；短线建议 C 权重高）：
- **A. 资讯面**：重仓行业/板块近期新闻、政策、研报标题 → LLM 判读情绪（偏正/中性/偏负 + 理由）
- **B. 重仓股舆情**：前十大重仓个股的公告/新闻/异动 → 聚合
- **C. 资金面/热度**（短线主力）：板块资金流向、成交热度、涨跌停、北向资金等**量化舆情**，纯数据无主观
- **短线取向**：C（资金/热度）> A/B（读新闻）——短线赚资金和情绪的钱，新闻滞后、噪音大、易被"利好出尽"反杀。

### Layer 3 · 精筛验风险（不可省）
短名单**逐只**拉 evidence 证据卡，校验：**52周位置**（越低越不追高）、**最大回撤**（历史腰斩幅度）、**RSI**（>70超买排除）、**夏普/卡玛**。杜绝"用收益排名冒充靠谱"。

## 3. 最终综合权重（用户定：按权重分配，可调）

两大层加权：**量化基本面 : 舆情**
- 稳健派 **70:30**（推荐）——基本面/趋势为主心骨，舆情做排雷+择时微调
- 均衡派 50:50
- 短线派 30:70——舆情/资金主导，波动大易反噬（≈用户已否掉的"半导体山顶"坑，慎用）
- **权重做成可调参数**（`--weights` 或配置），不同人/行情可改。
- 设计立场：即便短线，也建议 70:30 或 50:50，守住用户反复强调的"靠谱、高概率正收益"。

## 4. 数据来源（akshare）
- 全量收益榜：`fund_open_fund_rank_em`（已用，多类型合并去重）
- 单只净值/持仓/经理/评级：`fund_open_fund_info_em` 等（evidence 已用）
- 舆情待接：资讯类接口（新闻/研报）、资金流/北向/热度类接口（二期调研具体接口，记入 `docs/akshare-interfaces.md`）

## 5. 输出形态
- 新增 CLI：`quantfox deep-screen`（暂名），参数：`--horizon`(短/中/长)、`--weights`、`--per-theme`、`--top`
- 产出：分散短名单（跨板块）+ 每只的量化分/舆情分/风险校验 + 幸存者偏差与"过去≠未来、不保证盈利"免责
- 可选：复用 `report` 出可视化 HTML、`email` 发送

## 6. 诚实边界（必须保留）
- **幸存者偏差**：收益榜只含存续基金，清盘的已消失，历史收益天然偏高。
- **舆情噪音**：新闻滞后、易误导；短线尤其"利好出尽"，故 A/B 权重需克制。
- **无保证**：任何筛选都是候选与概率，非投资建议、不保证盈利。
- **短线真相**：追动能历史胜率仅 55-70%（实测超买态持有1-3月），赢平均2-9%、输一次-20~36%，必须配止损与小仓位。
