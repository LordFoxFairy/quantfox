# "收益诉求"用户旅程 SOP 设计（yield-seeker）

> 来源：2026-07-10 晚实战会话（黄金持仓复查 → 用户提出"1万1个月赚5千" → 全市场深筛 → 债基对照 → 008009 触发式方案）。
> 目的：把这条**端到端旅程**沉淀为 skill 增强项，交给下一个 code agent 打磨。
> 定位：**不建议新建第 8 个 skill**，建议作为 fund-screener 的"诉求入口"前置章节 + 若干引擎小改（C 系列工程项见文末）。理由：旅程的中后段（筛选/闸门/forecast/回测/舆情）fund-screener 与 fund-analyze 已覆盖，缺的是**前段（预期校准、风偏探测）和粘合层（假稳过滤、名实核对、时点纪律）**。

## 0. 触发语料

用户说："有什么基金值得入手" / "我想投资赚钱" / "帮我选收益高又稳的" / "1万一个月能赚X吗"。
共同特征：**诉求模糊 + 隐含收益预期未说出口或不现实**。直接跑 screen 端榜单 = 答错题（见 [[deep-fund-screening-means-full-market]] 教训）。

## 1. SOP 十步（本会话实证走通的完整链路）

### 第1步 · 预期校准（诚实红线前置，必做）
把用户隐含收益率**算出来当面点破**："1万月赚5千=月化50%=年化~13000%，全市场最强基金历史最疯单月+38~50%（十年一遇极值）"。
然后给**现实量级阶梯表**（本会话实测数字，打磨时可做成模板）：

| 去处 | 1万/年 | 风险 |
|---|---|---|
| 银行1年定存 | ~100元 | 零 |
| 货基 | ~130元 | 近零 |
| 政金债基 | ~300-360元 | 极端段回撤-2.8% |
| 增强债基 | ~500-1200元(不稳定) | 回撤-2% |
| 头部股基 | 中位+590/月 | 单月尾部-18%，历史腰斩 |

**要点：不点破就往下走 = 后面所有产出都会被用户用错误标尺评价（"感觉好拉"）。**

### 第2步 · 风险偏好探测（对话信号，不发问卷）
用户对第1步阶梯表和风险但书的**情绪反应就是偏好信号**：
- 嫌收益拉/"不如银行" → 锚定在幻想收益上，但通常真实风偏低 → 引导到"债基底仓+小卫星"
- 追问怎么进/接受回撤数字 → 风偏真高 → 给卫星仓纪律
- 本会话实证：用户先嫌拉，摊开确定性/收益交换后自己选了"高收益里最稳"——**探测结果决定第9步给几档方案**。

### 第3步 · regime 前置（已有）
`market-valuation` → 偏贵则 steady/pullback 风格（B2 已实现，照用）。

### 第4步 · 全类型深筛（小改）
`screen --type 全部`（不是默认股票型）——"稳"的答案可能在混合/债券型里。本会话 universe 19,943（全部）vs 1,074（股票型），头部混合型（008009 前身榜单里的 001412 等）正是从全类型口径进来的。

### 第5步 · 风险调整排序（B6 心智模型的落地管线，当前是手工的）
对 screen top-k 逐只 `evidence` 提取 夏普/卡玛/最大回撤/波动/估值分位 → **Pareto 排序（卡玛优先）**，不看裸收益。
本会话实证：收益王 006265（1年+276%）因回撤-65% 被卡玛排序正确淘汰，008009（夏普1.06/卡玛0.68）胜出。
**痛点：20 次 evidence 循环慢且有限流风险 → 见工程项 C1。**

### 第6步 · 假稳过滤（新增检查清单，本会话抓到两个真实反例）
榜单口径的"稳"有三类诈骗，精筛时逐项排查：
1. **迷你基净值脉冲**：回撤极小但年化波动异常高（如 014502：回撤-1%/波动12.8%/1年+30%"纯债"）= 大额赎回费一次性计入净值，不可复制。判据：`max_dd 极小 && ann_vol 与回撤量级严重不匹配`。
2. **债壳股胆**：类型标"债券型"但回撤 >10%（如 610108：信用债名义、回撤-23.8%，重仓可转债）。判据：`type 含'债' && max_dd < -0.10`。
3. **小样本虚高**：成立 <3 年的基金，forecast 条件化样本太少（如 015724"3月100%正收益"）。判据：`inception 距今 <3y 或条件化 n 过小` → 结论必须降权并明示。

### 第7步 · 估值闸门 + forecast 条件化 + 回测背书（已有，照框架 v13）
本会话实证了三者可以互相打架（008382：条件化分布好看 vs combo 回测扣成本净负）——**打架时按框架取观望，把分歧点写给用户**，不要挑对自己结论有利的那个。

### 第8步 · 舆情按"重仓板块"搜，不按基金名（名实核对，新增）
先持仓穿透再派调研：008009 名叫"高端装备"，实际=AI光模块+存储芯片+新能源。按名字搜舆情会全错。
**判据：搜之前必看 `profile.holdings.top`，按前十大聚类出 2-3 个真实主题再搜。**

### 第9步 · 输出三件套（格式沉淀为模板）
1. **分档方案**（按第2步探测结果给 1-3 档）：保守=债基底仓 / 平衡=7:3 / 进取=卫星分批。
2. **触发式分批进场卡**（四要素缺一不可）：**企稳信号**（如连续3日不创新低+2%阳线）/ **深值信号**（如距高点-25%）/ **止损**（首笔-10%）/ **整轮放弃条件**（基本面证伪清单）。落刀当天（单日-5%级）绝不给"现在买"。
3. **时点纪律**：周五15:00后下单与下周一14:59下单确认同一净值——**周末前夜下单=零收益纯亏信息权**；7日内赎回费1.5%；事件日历（CPI/FOMC/中报）标出决策点。

### 第10步 · 闭环（已有）
入围者 `watch add --note`（写清买入条件）+ 结论 `log-signal` 存档 + 报告 `report --analysis-file`。本会话产出：watch 7 只观测+1 持有，预测账本 #2(002611观望)/#3(008009观望)，报告 2 份。

## 2. 工程待办（C 系列，给 code agent）

### C1（P1）批量风险指标：screen 慢通道问题
- **场景**：第5步对 top-20 逐只 evidence，每只全量拉取（profile/持仓/评级都要），耗时数分钟且 akshare 限流风险。
- **建议**：新增 `quantfox metrics-batch <codes...>`（或 `screen --with-metrics`）：只拉净值序列算 夏普/卡玛/回撤/波动/估值分位 五列（跳过 profile/持仓），并发+重试+失败标记。影响：fund-screener。

### C2（P1）假稳检测 flags
- **场景**：第6步三类假稳当前靠人眼。
- **建议**：`evidence`（或 metrics-batch）输出加 `flags` 数组：`nav_spike_suspect`（回撤/波动不匹配，阈值如 |max_dd|<3% 且 ann_vol>8%）、`bond_equity_risk`（债券型且 max_dd<-10%）、`short_history`（<3年）。SOP 里写"带 flag 的候选必须向用户明示"。影响：fund-screener / fund-compare。

### C3（P2）名实不符提示
- **场景**：第8步，基金名与实际持仓主题背离（008009 案例）。
- **建议**：evidence 的 holdings 段加一行 `theme_guess`（前十大按行业聚类的主题猜测），与基金名不一致时 SOP 要求点明。可先纯 SOP（"搜舆情前必读前十大"），引擎实现放 P2。影响：fund-analyze / fund-screener。

### C4（P2）forecast 小样本警示
- **场景**：015724"3月100%正收益"是小样本伪象。
- **建议**：forecast 输出对 `n<200` 或基金年龄 <3y 的条件化分布自动附 `warning` 字段；SOP 规定此类数字必须打折表述。影响：fund-analyze / fund-screener。

### C5（P3）时点纪律 helper
- **场景**：第9步"周五晚下单=周一盲买"每次靠人算。
- **建议**：`quantfox next-confirm`（或 watch check 附带）：给定当前时刻 → 输出"现在下单的确认净值日"。纯日历+15:00 cutoff 逻辑，几行代码。影响：全部交易类 skill。

### C6（P3）预期阶梯表数据化（可选）
- **场景**：第1步阶梯表数字目前手填。
- **建议**：暂不做引擎（YAGNI）——把本文件第1步的表作为 SOP 模板文本维护，数字每半年人工校一次即可。

## 3. SOP 文本落点建议

- fund-screener SKILL.md 头部加"第0步：诉求校准与风偏探测"（本文件第1-2步内容压缩版）。
- 第6步假稳清单、第8步名实核对、第9步三件套模板 → 建议进 B7-② 计划中的**共享话术段**（7 skill 统一引用），避免各写各的。
- 触发式进场卡四要素格式可直接抄本会话 008009 报告的 risks_html 结构。

## 4. 本会话实证数据（打磨时可当测试样例）

- 全类型 screen universe=19,943；股票型=1,074；债券型=4,814。
- Pareto 胜出：008009（夏普1.06/卡玛0.68/回撤-48.7%/1年+166%/估值分位96.4%）。
- 假稳反例：014502（回撤-1.04%/波动12.79%/1年+30.3% 长期纯债）、610108（普通债券/回撤-23.81%）。
- 小样本反例：015724（2022成立，3月条件化 P正100%）。
- 证据打架案例：008382 条件化 20日 P正81%/中位+6.7% vs combo 回测 edge+2.2pp 但扣成本净-0.49%。
- 落刀案例：008009 于 7/10 单日-5.25%、距 6/30 高点-16.6%（高低切换兑现）。
